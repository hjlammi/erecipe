An important client of ours has an energetic grandmother, who really enjoys cooking. Due to personal reasons, though, she can't update her recipe collection manually anymore. She has asked YOU(!) to implement an electronic recipe book 'eRecipe' to easily manage all the precious family heirlooms.

Implement a simple eRecipe application using Python. The following criteria should be adhered:
1. Create a command line application containing classes for a recipe book and single recipes.
2. Parse grandmother's manually entered recipes from the file 'recipes.json' and save them into the applications internal memory.
3. Implement a REST API, which can be used to query recipes by ingredient. Grandmother recommends you to use Flask Library for this (http://flask.pocoo.org/), it is not enforced, however.

Please send your app back via email or e.g. a GitHub repository. We will forward it to grandma. You shouldn't really use more time than a few hours for this, since its for grandma's internal use only.

assignment.zip
├── README.txt      <- these instructions
└── recipes.json    <- Grandma's old, manually entered recipes

Erään tärkeän asiakkaamme mummo pitää kovasti kokkailusta. Hän ei kuitenkaan enää kykene ylläpitämään paperista reseptikokoelmaansa henkilökohtaisista syistä. Siksi hän onkin pyytänyt sinua toteuttamaan elektronisen reseptikirjan 'E-Resepti' perintöreseptiensä helppoon hallintaan.

Toteuta Pythonilla yksinkertainen reseptisovellus, joka toteuttaa seuraavat toiminnot:
1. Koodaa komentorivisovellus, joka kuvaa reseptikirjaa ja sen sisältämiä reseptejä sopivilla luokilla.
2. Hae mummon entiset muistiinpanot tiedostosta recipes.json, ja tallenna ne sovelluksen muistiin.
3. Toteuta reseptisovellukselle REST-rajapinta, josta mummo ja ystävänsä voivat hakea reseptejä ainesosan perusteella. Mummo suosittelee käyttämään tähän Flask-kirjastoa (http://flask.pocoo.org/), mutta mummo sallii muutkin vaihtoehdot.

Voit palauttaa vastauksesi sähköpostilla tai esim. Github-repositoriossa. Tehtävään ei kannata käyttää muutamaa tuntia enempää.
